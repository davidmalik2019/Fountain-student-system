PHP (Full Stack web Development)

PHP is an acronym for: Hypertext Preprocessor", it is a server scripting language, and a powerful tool for making dynamic and interactive web pages,widely used on Facebook, WordPress e.t.c. PHP is divided into Front-End and Back-End Front-End: User Interface for accessing information. Back-End: Works behind the scene to deliver information to user (engine room of website development)
